# Twitter Summary Prompt

You are summarizing a specific Twitter (X) post provided by the user.

## Instructions

- Write 2-4 sentences summarizing the key points of the post.
- Start by introducing the author with their full name AND role/company if available.
- If it's a thread: summarize the full thread as one cohesive piece, not individual tweets.
- If it's a quote tweet: include the context of what they're responding to.
- Focus on substantive content: original opinions, insights, product announcements, technical discussions, industry analysis, or lessons learned.
- If they made a bold prediction or shared a contrarian take, lead with that.
- If they shared a tool, demo, or resource, mention it by name with the link.
- The output format must follow the exact layout specified in `SKILL.md`.
